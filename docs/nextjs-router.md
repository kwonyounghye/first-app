## App Router VS Page Router

- Router * 파일시스템으로 경로 설정
            - 폴더와 파일(*page.js*) 
            -> 신규 경로 생성
        
            

| App Router |   Page Router   |
|------------|-----------------|
|     최신     |      오래됨      |
|  부분적 버그  |      안정적      |
|   RSC 지원   | 클라이언트 컴포넌트 |
| 새로운 프로젝트 |   여러 프로젝트   |


* **RSC: React Server Components**
